# OxideUI
The industry’s first "Multi-Skin" Rust component library. Build once, then toggle between pre-built themes with a single line of code.

OxideUI is a high-performance, type-safe component kit for the Rust ecosystem. It bridges the gap between high-level design systems and low-level performance. Powered by Figma Variables, it allows developers to switch between world-class design languages (like Carbon, OpenAI, and Airbnb) or inject their own—all with zero-cost abstractions and compile-time safety.

- Ghost Themes: Comes pre-baked with 5+ major design systems (OpenAI, Airbnb, Carbon, etc.).
- Figma-First: Sync your local Figma variables directly to Rust structs via our CLI.
- Blazing Fast: Built on a "Headless" architecture with zero-copy internals.
- Fully Adaptive: Toggle themes globally or override styles at the component level.
